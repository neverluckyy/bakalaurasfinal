# Phishing Examples Images

This directory contains screenshot images for the phishing examples in Module 1, Section 1: "Real-World Examples".

## Required Images

Please add the following image files to this directory:

### Pair 1: Bgs Security Quarantine
- `bgs-security-email.png` - Screenshot of the "Bgs Security: You Have Messages In Quarantine" email
- `bgs-security-website.png` - Screenshot of the fake sign-in page (enogp3wehr.pages.dev)

### Pair 2: Office365 Password Update
- `office365-email.png` - Screenshot of the "Operations Authentication / Office365 password update" email
- `office365-website.png` - Screenshot of the spoofed Microsoft login page (plast.toys.up-plastic.com)

### Pair 3: ShareFile Encrypted Message
- `sharefile-email.png` - Screenshot of the "Encrypted message / ACH transfer awaiting your review" email
- `sharefile-website.png` - Screenshot of the fake "Webmail Portal Access" login page (r2.dev)

## Image Specifications

- Format: PNG (recommended) or JPG
- Recommended dimensions: 1200-1920px width for best display quality
- File size: Keep under 2MB per image for optimal loading

## Usage

These images are referenced in the learning content via markdown image syntax:
```
![Alt text](/phishing-examples/filename.png)
```

The images will be displayed in the "Real-World Examples" section of Module 1, Section 1.
